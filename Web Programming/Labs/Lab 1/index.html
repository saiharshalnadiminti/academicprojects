• Why do we need a Uniform Resource Locator (URL)?

    • Ans. URL is used in order to allow clients to request particular resources from the server.

  • Give an example of Uniform Resource Locator (URL) and specify all components of an URL.

    • Ans. Example: http://www.funwebdev.com/index.php?page=17/#article
            Components of an URL:
            1) Protocol
            2) Domain
            3) Path
            4) Query String
            5) Fragment

  • What is the difference between a static and a dynamic web site?

    • Ans. Static websites are the websites that consists only of HTML pages that look identical for all users at all times.
             Dynamic websites are the websites in which the page content is being created at run-time by a program created by a programmer; this page content can vary for user to user.

  • Why do we need Domain Name System (DNS)?

    • Ans. Since we humans cannot remember IP addresses of the websites DNS helps us to access websites with the domain names instead of the IP adresses.

  • What is DNS address resolution?

    • Ans. The process in which the domain name system provides a mechanism for a software(browser) to discover the numeric IP address of a website this process is referred as DNS address resolution.

  • What is the function (purpose) of Hypertext Transfer Protocol (HTTP)?

    • Ans. HTTP is used to establish a connection with the server and send HTML pages back to the user's browser.

  • Give some examples of HTTP request methods.

    • Ans. Examples of HTTP request methods:
            1) GET
            2) POST
            3) HEAD
            4) PUT
            5) CONNECT

  • Define a Web Server.

    • Ans. A web server is server software, or hardware dedicated to running this software, that can satisfy client requests on the World Wide Web.
