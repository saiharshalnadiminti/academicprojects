Elements of HTML5

  1. Heading Element:
    • Syntax:
        <h1> This is heading 1 </h1>
        <h2> This is heading 2 </h2>
        <h3> This is heading 3 </h3>
    • Output:

       This is heading 1

         This is heading 2

          This is heading 3

  2. Paragraph Element:
    • Syntax:
        <p> Text inside a paragraph Element </p>
    • Output:

         Text inside a paragraph Element

  3. Line Break Element:
    • Syntax:
        <p>a<br>p<br>p<br>l<br>e<br>s</p>
    • Output:

       a
       p
       p
       l
       e
       s

  4. Blockquote Element:
    • Syntax:
        <blockquote>" Text inside a blockquote Element "</blockquote>
    • Output:
       "This is a paragraph"
  5. Phrase Elements:
    • Syntax:
        <b>bold</b> text.
        <en>empasized</en> text.
        <i>italicized</i> text.
        <mark>mark</mark> text.
        <small>small</small> text.
        <strong>strong</strong> text.
        <sub>sub</sub> text.
        <sup>sup</sup> text.
    • Output:
        bold text.
        empasized text.
        italicized text.
        mark text.
        small text.
        strong text.
        sub text.
        sup text.
  6. Lists:
      7. a) Lists Unordered
    • Syntax:
        <li> example a </li>
        <li> example b </li>
    • Output:
    • example a
    • example b
      b) Lists Ordered
    • Syntax:
       <ol>
           <li> example a </li>
           <li> example b </li>
      </ol>
    • Output:
    1. example a
    2. example b
      c)Description list
    • Syntax:
        <dl>
           <dt> Triangle </dt>
              <dd> Pyramid </dd>
           <dt> Square </dt>
              <dd> Cube </dd>
        </dl>
    • Output:
           Triangle
            Pyramid
           Square
            Cube
  7. Special Characters:
    • Syntax:
    "copyright" :&copy <br>
    "less" :&lt <br>
    "greater" :=&gt <br>
    "ampersand" :=&amp <br>
    • Output: 
            "copyright" :© 
            "less" :< 
            "greater" :=> 
            "ampersand" :=&
  8. Anchor Element
      9. a)Absolute Link
    • Syntax:
    Link to a different website: <a href="http://www.google.ca">www.google.ca</a>
    • Output: 
      •    Link to a different website: www.google.ca
      b)Relative Link
    • Syntax:
    Link to a different page on the same website: <a href="../Lab1/index.html">Lab1</a>
    • Output: 
      •     Link to a different page on the same website: Lab1
      c)E-mail hyperlink
    • Syntax:
    Link to mail program connected with broswer or a message is displayed: <a href="mailto:nadimintiharshal1@yahoo.com">nadimintiharshal1@yahoo.com</a>
    • Output: 
      •     Link to mail program connected with broswer or a message is displayed: nadimintiharshal1@yahoo.com

  9. Meter Element:
    • Syntax:
    <meter value="100" min="0" max="100">Max 100 </meter> <meter value="50" min="0" max="100">Max 50 </meter> <meter value="10" min="0" max="100">Max 0 </meter>
    • Output:

        Max 100     Max 50     Max 0

  10. Progress Element:
    • Syntax:
    <progress value="700" max="1000">700</progress>
    • Output:

        700
