(i) What are main benefits of using CSS?

a) The degree of formatting control in CSS is significantly better than that provided in HTML.
b) CSS-driven sites are more accessible.
c) CSS can be used to adopt a page for different output mediums.


(ii) What are the different parts of a CSS style rule?

a) Selector b) Declaration


(iii) What are an element selector and grouped element selector in CSS? Provide an example of each.


Element Selector: Selects all instances of a given HTML element example: *
Grouped Selectors:Selects multiple things example: p,



(iv) What are class selectors in CSS? What are id selectors in CSS?

A class selector allows you to simultaneously target different HTML elements regardless of their position in the document tree.
An id selector allows you to target a specific element by its id attribute regardless of its type or position.



(v)What are contextual selectors in CSS?

A contextual selector contextual selector (in CSS3 also called combinators) allows you to select elements based on their ancestors, descendants, or siblings.



(vi) What are the three cascade principles used by browsers when style rules conflict?


The three cascade principles are:
1) Inheritance
2) Specificity
3) Location
